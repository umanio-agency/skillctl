# skills-cli

A CLI tool to manage your personal Claude skills library across projects.

> Status: pre-v1, in active development.

## Why

You probably have a personal repo where you collect Claude skills you've built. Reusing them in a new project means manually duplicating folders, and any improvement you make in a project is hard to push back to the source repo. `skills-cli` makes that round-trip painless.

## Features (planned)

- Link any GitHub repo as your skills library — no fixed folder structure required (skills are detected by `SKILL.md`).
- List all skills found in the linked repo.
- Multi-select skills to install into the current project.
- Detect local changes to installed skills and push them back to the source repo.
- Detect new skills created in a project and offer to add them to the source repo.
- Fork a modified skill into a new skill instead of overwriting the original.
